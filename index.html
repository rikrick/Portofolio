MakeMeUpzz

Makemeupzz is web-based online makeup and skin care selling application with basic design. On the website, we can log in as admin to add products and we can also log in as customers to buy cosmetics. This web is made using html as the frontend, c# as the backend, sql for the database and also uses design pattern concepts such as controller, factory, repository and handler. This web was created using the ASP.NET framework.
