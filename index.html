Project

Hi-Toyz

Hi-Toyz is a web that will be used as a media introduction for companies engaged in the sale of collection-based toys. On the web there is a map that visitors can use to find out that Hi-toyz also has an offline store and also this web has a purchase feature. This web was made to complete the final project of the HCI (Human Computer Interaction) course so that it is only in the form of frontend, which is only with html, css and javascript.

MakeMeUpzz

Makemeupzz is web-based online makeup and skin care selling application with basic design. On the website, we can log in as admin to add products and we can also log in as customers to buy cosmetics. This web is made using html as the frontend, c# as the backend, sql for the database and also uses design pattern concepts such as controller, factory, repository and handler. This web was created using the Microsoft .NET framework.
